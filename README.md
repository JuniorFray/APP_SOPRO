# Sopro

> "O sussurro certo. No lugar certo."

App de memoria fisica contextual para Android (iOS-ready).
Ver CLAUDE.md para contexto completo do projeto.

## Setup rapido
```
flutter pub get
flutter run
```

## Estrutura
```
lib/
  core/          - Constantes, tema, utils
  data/          - Banco, modelos, repositorios
  domain/        - Entidades, casos de uso
  presentation/  - Telas e widgets
  infrastructure/- GPS, BLE, ML, notificacoes
```
